Safari, свой среди браузеров или не такой как все?
Создание, развитие, на чем базируется.


Safari — веб-браузер, разработанный корпорацией Apple и входящий в состав macOS и iOS. По состоянию на февраль 2019, занимает второе место по числу пользователей в мире.


История

Safari основан на свободно распространяемом коде движка WebKit. Данный браузер создавался, когда подходил к концу срок договора Apple с Microsoft о разработке Internet Explorer для Mac OS X. Вскоре после появления Safari работа над Internet Explorer for Mac была прекращена.


Развитие

В июне 2007 года состоялся релиз тестовой версии Safari 3.0 для операционных систем Windows XP и Windows Vista. Первое время сотрудники Apple утверждали, что их браузер загружает и отображает страницы быстрее Internet Explorer и Mozilla Firefox. В действительности подобные заявления подвергались сомнениям из-за тестов производительности, которые были разработаны компанией VeriTest. По некоторым данным они давали браузеру Safari преимущество перед конкурентами. Позже состоялся релиз версий 3.1 и 3.1.1. Стоит отметить, что версии браузера для Mac OS и MS Windows практически эквиваленты в плане производительности, в отличие от Firefox, где производительность версии для семейства операционных систем Linux лучше. В феврале 2009 была выложена в свободный доступ публичная бета-версия четвертого издания Safari для операционных систем Mac OS и Windows. Из новых особенностей была представлена услуга Top Sites. С ее помощью пользователь получает быстрый доступ к наиболее часто посещаемым сайтам, миниатюры которых располагаются полукругом. Также добавили возможность просмотра истории посещений в режиме Cover Flow. В ноябре 2010 года состоялся релиз пятой версии Safari. В ней был добавлен режим Reader, который отображает на странице только текст, а также были изменены настройки поиска. В числе иных изменений: инструментарий для разработчиков, поддержка расширений (только для Windows), расширенная система DNS и поддержка стандарта HTML 5, решены проблемы с Flash.
